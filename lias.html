LoRaWAN Project Summary

fr: La couche physique LoRa permet des communications longue distance sans fil. Un dispositif LoRa envoi un message très court vers une passerelle, qui l'envoi vers un serveur d'application où un utilisateur peut récupérer les données sur l'internet. Nous souhaitons minimiser la puissance consommé par le dispostif et assurer la précision des messages envoyés. Nous allons étudier plusieurs variations d'une architecture LoRa: un dispositif qui communique avec The Things Network par des passerelles déjà déployés, un dispositif et une passerelle qui utilisent les applications LoRa Fabian, et une passerelle qui met en oeuvre les applications fournies par LoRa Server Project.

en: The LoRa physical layer allows for long distance wireless communications. A LoRa device sends a very short message to a gateway, which in turn sends it to an application server where a user may view the data on the web. We wish to minimize the power consumed by the device and maximize the precision of the messages that are sent. We will study several variations on the LoRa architecture: a device that communicates with The Things Network via previously deployed gateways, a device and a gateway that utilize the LoRa Fabian applications, and a gateway using the LoRa Server Project software.
